## Proyecto 3

Implementación de un programa en Java que permite **ensamblar y desensamblar** programas escritos para la arquitectura Hack:

- **Ensamblador:** Traduce código en lenguaje ensamblador Hack (`.asm`) a binario de 16 bits (`.hack`)
- **Desensamblador:** Traduce un archivo binario Hack (`.hack`) de vuelta a lenguaje ensamblador (`.asm`)

Para más detalles sobre compilación, uso y documentación técnica, consultar la carpeta `HackAssembler/docs/`.
